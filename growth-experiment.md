# 可验证增长实验（开源贡献增长）

## 目标
验证 Orion Nexus 是否具备“持续吸引外部贡献”的起步信号。

## 实验设计（14天）
- 方案A：仅发布 README + Roadmap
- 方案B：README + `good first issue` + 贡献奖励说明
- 渠道：GitHub + 开发者社区帖（同等曝光）

## 指标
- 主指标：外部 PR 数（unique contributors）
- 次指标：star 增量、issue 互动率、文档访问到 fork 转化率

## 样本量与阈值
- 最低样本：曝光点击 >= 1000（A/B 各 >= 500）
- 成功阈值：
  - B 组外部贡献者数 >= A 组 1.5x
  - 且 B 组 PR 合并率 >= 30%
  - 且新增 star/fork 转化率 >= 5%

## 判定
- 达标：保留 B 组机制并扩展月度挑战赛
- 未达标：优化 onboarding 与任务颗粒度后重测
