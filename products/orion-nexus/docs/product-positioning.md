# 产品定位

## 愿景
打造“可比较、可复现、可协作”的美股量化开源研究底座，持续汇集并超越 GitHub 最佳实践。

## 用户
- 独立量化研究者
- 开源贡献者
- 小型研究/资管技术团队

## 差异化
- 策略聚合评测协议（统一输入/输出）
- 强制风险指标与风控闸门
- CLI 优先，开发者友好
- 可演进到 broker 适配的自动化交易

## 路线图
- v0.1：离线评测+HTML报告（已完成）
- v0.2：多数据源分钟级演示+交易成本建模
- v0.3：Paper Trading 适配（Alpaca/IBKR/Tradier）
- v1.0：社区策略榜单与评测挑战赛
