# 从回测到模拟盘/实盘迁移步骤（含风控红线）

1. **回测阶段**
   - 固定样本区间与参数
   - 输出策略报告（收益+风险）
2. **走样本外验证**
   - 至少追加 3-6 个月 OOS 测试
3. **模拟盘（Paper Trading）**
   - 接 broker paper 账户
   - 打开 pre-trade 风险闸门
4. **小资金实盘**
   - 单笔风险 <= 0.25%-0.5% 净值
   - 日内回撤 >2% 立即停止新开仓
5. **扩规模**
   - 连续 8-12 周稳定后再提高仓位

## 风控红线
- 未通过 OOS 验证不得实盘
- 不得绕过风险闸门直接下单
- 出现异常成交/滑点飙升时自动熔断
