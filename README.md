# Passive Income Lab

目标：30天内，围绕可落地、合规、可盈利的数字化被动收益方向，完成“调研 -> 原型 -> 验证 -> 迭代 -> 公开沉淀”。

## 原则
- 先做能跑通的小闭环，再扩张。
- 只做可验证的事，不刷无意义提交。
- 安全优先：不执行未知脚本、不跑可疑二进制。
- 合规优先：不碰灰产、侵权、欺诈。

- `results.md`：阶段成果与指标
